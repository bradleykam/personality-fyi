About Personality.fyi

Updated 2026-04-22

What this site is

Personality.fyi is a free MBTI personality platform. You take a 60-second test (or pick your type manually), and the site becomes a workspace: a type profile, career fit, compatibility with any other type, and an AI advisor that answers questions about any of the 16 types.

There are a lot of personality sites out there. Most of them fall into one of two bad buckets: horoscope-style "your type is special!" content, or over-hedged therapy-speak that refuses to say anything concrete. This site does neither. The goal is specific, honest analysis you can actually use — at work, in relationships, and in the decisions that come up when you're trying to figure out what to do next.

How the test works

The 60-second test analyzes your writing style against patterns for each of the 16 Myers–Briggs types. You paste or type a short passage, and we score it on four axes: Introvert/Extravert, iNtuitive/Sensing, Thinking/Feeling, Judging/Perceiving. The result is a 4-letter type code plus a confidence score. It’s not an official MBTI assessment — the Myers–Briggs Foundation is the only group that licenses the official test — but the mapping is based on well-documented cognitive-function patterns that researchers have validated for decades.

If the result doesn’t match what you expected, pick your type manually. You know yourself better than any short passage of text.

How the content is written

Every type guide, compatibility page, and career fit article on this site was written with the same rules: lead with the concrete answer, use specific examples, name both the strengths and the shadow behaviors of each type, avoid therapy-hedging and horoscope framing. If a type has a pattern that can be uncomfortable to read, the page says so. The point is to be useful, not flattering.

The AI-powered features (Career Advisor, Ask a Type, Compatibility analysis) use Anthropic’s Claude model with explicit instructions to be direct and specific, not generic.

On accuracy and MBTI as a framework

MBTI is a useful framework, not a scientific classification. The four-letter codes are a pattern-matching shorthand that captures real differences in how people think and behave — but individuals don’t fit cleanly into 16 boxes, and any single type description will feel partially wrong. Use it as a lens for self-reflection and communication, not a personality verdict.

If you want academic rigor on cognitive differences, read about the Five-Factor Model (Big Five) — it’s more scientifically defensible. But MBTI is a better conversation starter, and that’s ultimately what this site is for.
